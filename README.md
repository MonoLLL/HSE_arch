# Проектирование архитектуры программных систем
**Тема: Разработка системы для коммуникации между учебным офисом и студентами**

## Краткое опимание
Темой проекта является разработка информационной системы для оперативного предоставления студентам университета справочной информации, связанной с организацией учебного процесса.
Главная задача системы - снизить нагрузку на учебный офис, устранить необходимость в лишних обращениях студентов и предоставить быстрый и удобный доступ к ответам на типовые вопросы.

Проект включает:
- базу знаний (FAQ, события, сроки);
- админ-панель для сотрудников учебного офиса;
- модуль интегрированного поиска;
